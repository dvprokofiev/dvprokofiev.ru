# dvprokofiev.ru

### или же исходный код моего веб-сайта

Использует суперсилы **[Hugo](https://github.com/gohugoio/hugo)**

## Разработка

Запуск локального сервера с черновиками:
`hugo server -D`

Сборка для продакшена:
`hugo --gc --minify`
